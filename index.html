One-time setup (free):

  1. Go to console.cloud.google.com and make a project (any name).
  2. Search “Google Drive API” and click Enable.
  3. Open APIs & Services → OAuth consent screen, choose External, fill the app name + your email, and add yourself as a Test user.
  4. Open Credentials → Create credentials → OAuth client ID → Web application.

  6. Create it, copy the Client ID (ends in .apps.googleusercontent.com), and paste it in the box above.
